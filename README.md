# Simple Interest Rate & Utility Scripts

This repository contains a few beginner-friendly Python scripts for learning and practicing basic programming concepts.

## Files

- `input.py` – Handles basic simple interest.
- `pycalculator.py` – A simple calculator for basic arithmetic operations.
- `temp_converter.py` – Converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Features

- Simple calculator functionality
- Temperature conversion between different scales
- Easy-to-read and beginner-friendly Python code

1. Make sure you have Python 3 installed.
2. Clone the repository or download the files.
